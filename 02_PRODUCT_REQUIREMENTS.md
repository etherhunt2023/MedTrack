# Product Requirements

## Vision
A unified platform to track medicine purchases across pharmacies.

## User Stories
- As a user, I can record orders.
- As a user, I can upload invoices.
- As a user, I can view spending analytics.
- As a user, I can track inventory and expiry.
- As a user, I can compare historical prices.

## Acceptance Criteria
Every feature must include validation, loading, error and success states.
